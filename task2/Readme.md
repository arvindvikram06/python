# Real-Time Chat Server

A lightweight real-time chat server built with Python using WebSockets and AsyncIO. Supports chat rooms, private messaging, and persistent message history.

---

## Features

- Multiple chat rooms
- Real-time room messaging
- Private messaging between users in a room
- Live user list per room
- Chat history (room and private)
- Persistent storage using SQLite
- Asynchronous and scalable server

---

## Tech Stack

- Python 3
- AsyncIO
- websockets library
- SQLite
- JSON-based WebSocket protocol
- Frontend (React / HTML / JS) using WebSocket client
